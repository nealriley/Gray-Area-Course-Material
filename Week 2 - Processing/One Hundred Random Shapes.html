One Hundred Random Shapes

Your assignment is to write a program that uses a loop to draw 100 random shapes to the screen. Each time the screen is redrawn, the shapes will have some random difference. It could be location, color, size, opacity, or whatever. Your program may use 2d or 3d graphics. Your program doesn't have to look like mine above, it's exact appearence is entirely up to you. If you want, you could number each shape to show that there really are 100 shapes on the screen. Have fun and be creative!

Steps to completing this assignment

  1. Start a new program in Processing. Do not name your program 100Shapes or any name that starts with a number. Java does not allow program names to start with anything other than a letter of the alphabet.
  2. Write void setup() and void draw() functions.
  3. In draw(), write a loop that runs 100 times. Inside that loop place your code that generates one random shape.
  4. If you are using 3d graphics be sure to translate back after you draw your shape. For instance, my code looked like: 
    
  translate(xOffset,yOffset,zOffset);
  box(50);
  translate(-xOffset,-yOffset,-zOffset);
    where xOffset,yOffset, zOffset and randomSize are variables that I declared and assigned random values.

    Note: to keep your shapes from being translated outside the viewable region, have the zOffset be a random negative number. Translating in the negative z direction pushes the shape "away from you."

  5. Don't forget to put your name, class and assignment in comments.
