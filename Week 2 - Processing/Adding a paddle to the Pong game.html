Adding a paddle to the Pong game



Press '4' or '6' to move paddle left or right
You may need to click on the applet to give it the focus

We're now ready to add a paddle to the pong game.

  1. Start by writing a Paddle class. It should have functions that show() it to the screen and move() it to the left and right
  2. Create a new Paddle() and draw it to the screen
  3. Use an if statement in draw() to allow the user to move the paddle
  4. Modify the bounce() to use get() (like in the tron game) to see if the ball is going to hit something that is not black.

Extensions

Once you get the basics working, you can add extra features to your Pong Game. For example:
  1. Change the size or color of the ball with each bounce
  2. Keep score
  3. Allow the user to move the paddle up and down as well as side to side
  4. Change the angle of the bounce to depend on the part of the paddle that the ball hits.
  5. Add "bricks" to make a Break-Out style game
  6. Add gravity to make a Rebound style game
  7. Add a second player or a computer opponent
  8. Make a Pong clock--lock at this processing clock to get ideas.
Be creative, your Pong game doesn't have to look or work like any other.
