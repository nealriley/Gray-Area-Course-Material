Rotating Cube



click and drag to change position of light

In addition to the 2 dimensional graphics functions you've used, Processing also has a set of 3 dimensional graphics functions. In this assignment we'll use the box() to draw a cube.

Steps to completing this assignment

  1. Start a new program in Processing. Include the functions void setup() and void draw()
  2. In void setup() you will need to call the 3 argument version of size(), e.g. size(300, 300, P3D); since we will be using the 3d functions.
  3. In draw() you'll use the following functions in this order:
    • background()
    • translate() moves the coordinates where the box will be drawn
    • rotateX() rotates the box around the X axis
    • rotateY() rotates the box around the Y axis
    • box() draws the rotated and translated box
    Experiment with different arguments to get a feel for how they effect the box.
  4. You'll need someway to rotate the cube. It could rotate on its own, or you could add something like mouseDragged() with mouseX and mouseY used to change the amount of rotation. Whatever method you choose, since the amount of rotation will be changing, you'll will probably need a couple variables to store the amount of rotation in the x and y directions.
  5. You may want to use pointLight() to make a 3 dimensional lighting effect.
  6. These are only suggestions, feel free to modify this program as you wish.
