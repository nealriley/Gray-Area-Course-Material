Magic Eight Ball


This assignment asks you to write an applet that replicates the classic Magic 8 Ball toy from the 1960’s. Your program (like the original 8 Ball) will not really pay any attention to the user’s question—it will simply give one of several "canned" answers. The original Magic 8 Ball had these answers printed on the sides of two eight sided dice. When it was turned upside down, the dice would float to the top and one side of one die would be shown on the answer window. Here are the original 16 answers:

outlook not so good
don't count on it
my sources say no
without a doubt
reply hazy, try again
it is certain
my reply is no
as I see it yes
most likely
you may rely on it
cannot predict now
better not tell you now
very doubtful
outlook good
yes definitely
concentrate and ask again

You may use these answers or make up your own.

Arrays

For this assignment you are required to use an array of strings to store the answers. You can declare and intialize an array of strings with code like:


String[] answers = {"yes", "no", "maybe" };

Remember that arrays in are zero based:
println(answers[1]);
would print no.

Program Requirements

Don't name your program 8ball! Java doesn't allow names that start with digits. (If you do name your program illegally, processing automatically puts an underscore in front of the name to make it legal, but school loop won't accept files that begin with an underscore.)

Your program will have two modes: an "answer" mode and a "question" mode. You will need a boolean to keep track of which mode you are in.
