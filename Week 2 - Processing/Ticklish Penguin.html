Ticklish Penguin

Your assignment is to write a program that moves something in a random walk. Click on the penguin to see it move in a random walk. Your program doesn't have to look or work like my example. Any sort of random walk is fine. Check the samples of student work at the bottom for some ideas. Have fun and be creative!

Some examples from the "Real World"

Programmers create software using random walks to help researchers model and study many real world phenomena.

  • Here's an applet that demonstrates Brownian Motion, the random movement (closely related to a random walk) of particles suspended in a liquid or gas which was described by Einstein.
  • Here's another applet that shows that a gambler will almost certainly lose all his money in the long run, a phenomena called Gambler's Ruin.
  • This applet uses a random walk to simulate chemotaxis which I think means "animals foraging for food"
  • This applet uses a random walk to test different systems of investing in the stock market.

Steps to completing this assignment

  1. While you could use your picture from the penguin assignment, you may find it easier to move a simpler object like an ellipse or a point
  2. You could also move an image, see the image() example to get started.
  3. Once you've choosen what picture or object you want, make sure you can see it on the screen.
  4. Then make sure you can move your image or picture by changing it's x and y coordinates.
  5. Since the x and y coordinates of the picture or object will be changing, we will need variables for each
  6. In void draw(), change the variables that hold the x and y coordinates a small amount using random
