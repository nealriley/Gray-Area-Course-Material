Background:

The Etch-A-Sketch is a plastic tablet that lets you make designs by turning knobs that control a point (called a stylus). The left and right knobs control horizontal and vertical rods, moving the stylus where the two rods meet. When the stylus moves, it scrapes the screen, which is coated with a mixture of aluminum powder and plastic beads, leaving the line you see. The first Etch-A-Sketch was sold in 1960. Through television advertising it went on to become one of the most popular toys ever made.

In this assignment you will create an electronic version of the Etch-A-Sketch. This program will give you practice with if statements and keyboard input.

Since this is a larger program than the previous assignments, you may want to complete it in steps.

Step One:

It is a good idea to start by writing the code that allows you to "draw". We can use the point() function to draw one dot at a time on the screen. The function takes two integer arguments--the x and y coordinates of the point.

Here is an outline of the basic program. Your job is to translate the outline into computer code. You should then have a working program that will form the beginning of this assignment.

  1. Declare two integer variables for the X & Y coordinates of the stylus and initialize them to the center of the screen
  2. Write a setup() function that
    1. sizes the screen
    2. sets the background()
    3. sets the initial stroke() color
  3. Write a draw() function and leave it empty for now
  4. Write a keyPressed() function with the following if statements
    1. if the key pressed means left
      1. decrease the X coordinate by one
      2. draw a point at the current coordinates
    2. if the key pressed means right
      1. increase the X coordinate by one
      2. draw a point at the current coordinates
    3. if the key pressed means up
      1. decrease the Y coordinate by one
      2. draw a point at the current coordinates
    4. if the key pressed means down
      1. increase the Y coordinate by one
      2. draw a point at the current coordinates
