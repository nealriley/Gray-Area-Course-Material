User Input


Left lick and drag the mouse to make a design
Right click and drag to generate symmetrical reflections
Press any key to erase and start again

In this assignment you will write a program that takes user input from the keyboard and mouse to allow the user to interact with your program. Your program doesn't need to appear the same as the above example, check the links to sample student work below for ideas. Have fun and be creative.
    Program Requirements
  • Your finished program will need at least two functions void setup() and void draw().
  • It will also need to respond to keyboard and mouse input.
  • There will have to be at least two if statements, one for the mouse and one for the keyboard.

Optional--Put your program on an Android phone or tablet

Processing has an Android mode that makes it very easy to put your program on your phone or tablet. You'll need to make a few changes to your program though:
  1. Don't use the size() function. Every phone and tablet has a different size screen. To find the size call width and/or height.
  2. On your Android device, go to Settings | Security and choose Unknown sources - allow installation of non-Market apps
  3. Connect your Android device to your computer using a USB cable. You may need to install the appropriate driver.
  4. Start processing. Enable Android mode. Choose Sketch | Run on device. Processing should install the program and start it running on your phone or tablet.

Note: If you want to do this at home, you'll need to install the Android SDK and the USB drivers for your phone. You can find instructions here
